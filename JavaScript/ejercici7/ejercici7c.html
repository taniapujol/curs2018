El factorial de un número entero n es una operación matemática que consiste en multiplicar todos los factores n x (n-1) x (n-2) x .. x 1. Así, el factorial de 5 es igual a: 5! = 5x4x3x2x1=120 Utilizando la estructura for, crear un script que calcule el factorial de un número entero.
